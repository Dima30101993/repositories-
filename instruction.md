git --version - узнать версию программы
git init - начало отслеживания изменений в папке
git status - показывает информацию о файлах
git add - добавление файла в фиксацию и сохранение
git commit -m "text" - коментарий к сохранению
git branch - проверка наличия веток
git branch *имя новой ветки* - создание новой ветки
git branch -d *имя ветки* - удаление ветки
git checkout *имя ветки* - переход на другую ветку
git log - журнал изменений
git log--graph - лог коммитов с визуализацией между ними
git checkout _____(указать первые четыре символа комита) - переход на нужную версию
git clone ссылка на удаленный репозиторий - клонирование и загрузка чужого репозитория
cd название папки - поменять директорию
git pull - загружает версию с удаленного репозитория в локальный, делает слияние файла
git push - отправление изменений на удаленный репозиторий
